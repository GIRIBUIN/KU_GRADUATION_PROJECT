# 화면 흐름

## 1. 전체 탭 구조

하단 탭은 4개로 구성한다.

```text
홈 / 목록 / 관리 / 설정
```

| 탭 | 역할 |
| --- | --- |
| 홈 | 오늘 할 일, 마감 임박, 요약, 동기화 상태 확인 |
| 목록 | 전체 task 조회, 검색, 필터, 정렬 |
| 관리 | 태그, 폴더, 삭제된 task, 제외된 e-campus 항목 관리 |
| 설정 | 계정, 자동 동기화, 알림, 보안, 위젯 설정 |

## 2. 홈 화면

![홈 화면](../assets/mockups/01_home_subtasks.png)

### 주요 구성

- 오늘 날짜
- 동기화 아이콘
- 설정 진입 아이콘
- 오늘 할 일 개수
- 마감 임박 개수
- 미완료 개수
- 오늘 할 일 섹션
- 마감 임박 섹션
- 마지막 동기화 상태
- 개인 task 빠른 추가 버튼
- 하단 탭 바

### 표시 정책

- 홈은 전체 task 목록이 아니라 오늘 할 일과 마감 임박 task 중심으로 보여준다.
- 마감 임박 기본 기준은 오늘 포함 3일 이내이다.
- 마감 임박 기준 기간은 설정에서 변경할 수 있다.
- 오늘 할 일에 이미 표시된 task는 마감 임박 섹션에서 중복 표시하지 않는다.
- sub task가 있는 parent task는 `서브 작업 완료 수 / 전체 수`와 진행률 바를 표시한다.
- 홈에서는 sub task 목록을 직접 펼치지 않는다.

## 3. 목록 화면

![목록 화면](../assets/mockups/02_task_list.png)

### 주요 구성

- 검색 입력
- 필터 칩
- 정렬 메뉴
- task 리스트
- parent task 펼치기
- sub task 체크리스트 일부 표시
- 개인 task 추가 버튼
- 하단 탭 바

### 필터

- 전체
- e-campus
- 개인
- 미완료
- 완료
- 삭제됨

### 정렬

- 마감일순
- 우선순위순
- 생성일순
- 태그별

## 4. 동기화 결과 화면

![동기화 결과 화면](../assets/mockups/03_sync_result.png)

### 주요 구성

- 동기화 결과 요약
- 가져올 항목
- 업데이트 후보
- 가져오지 않을 항목
- 선택 항목 가져오기 버튼

### 분류 정책

`가져올 항목`에는 다음 항목을 표시한다.

- 신규 e-campus 항목
- 업데이트 후보 항목

`가져오지 않을 항목`에는 다음 항목을 표시한다.

- 이미 가져온 항목
- 완료된 항목
- 삭제된 항목
- 제외된 항목

사용자가 선택한 항목만 앱 task에 반영한다.

## 5. 작업 상세/수정 화면

![작업 수정 화면](../assets/mockups/04_task_edit.png)

### 주요 구성

- 제목
- 출처 정보
- 마감일
- 우선순위
- 태그
- 알림
- sub task 목록
- sub task 추가
- 메모
- 저장
- 삭제

### 정책

- 개인 task와 e-campus task 모두 같은 상세/수정 화면을 사용한다.
- e-campus task도 제목, 마감일, 태그, 우선순위, 알림, 메모를 수정할 수 있다.
- e-campus task는 화면 내에서 출처만 표시한다.

## 6. 관리 화면

![관리 화면](../assets/mockups/05_management.png)

### 주요 구성

- 태그 목록
- 태그 추가
- 폴더 목록
- 폴더 추가
- 삭제된 작업
- 가져오지 않을 e-campus 항목

### 정책

- 태그와 폴더는 관리 탭에서 생성, 수정, 삭제한다.
- 삭제된 task는 복구하거나 완전 삭제할 수 있다.
- 제외된 e-campus 항목은 다시 가져올 수 있도록 관리한다.

## 7. 설정 화면

![설정 화면](../assets/mockups/06_settings.png)

- 위젯 설정

## 8. 주요 이동 흐름

### 개인 task 생성

```text
홈 또는 목록
→ 추가 버튼
→ 작업 생성 화면
→ 저장
→ 홈 또는 목록 반영
```

### e-campus 동기화

```text
홈
→ 동기화 아이콘
→ 로그인 또는 저장된 계정으로 동기화
→ 동기화 결과 화면
→ 선택 항목 가져오기
→ 홈/목록 반영
```

### task 수정

```text
홈 또는 목록
→ task 선택
→ 작업 상세/수정 화면
→ 저장
→ 이전 화면 반영
```

### 태그/폴더 관리

```text
관리
→ 태그 또는 폴더 선택
→ 생성/수정/삭제
→ 목록 필터 또는 task 편집 화면에서 사용
```
